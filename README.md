# heart-disease-prediction

## Kaggle Competition
🔗 https://www.kaggle.com/competitions/playground-series-s6e2

Heart Disease Prediction (Kaggle)

This project demonstrates an end-to-end data science workflow applied to a large-scale healthcare dataset (600k+ observations) to predict heart disease risk. 
I combined exploratory data analysis with both interpretable (logistic regression) and advanced machine learning models (XGBoost) to evaluate performance and extract meaningful insights. 
Beyond model building, I also performed data cleaning, Exploratory data analysis, hyperparameter tuning and critically comparing results—highlighting why a simpler, interpretable model can outperform more complex approaches in structured data settings.
